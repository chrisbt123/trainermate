What TrainerMate does

TrainerMate acts as a careful admin assistant. It helps with repetitive checks and link assignment, but it does not replace your responsibility to confirm course details before delivery.

1

Course checks

Check assigned courses from supported provider portals and keep a local dashboard of what needs attention.

2

Zoom link assignment

Create, reuse or verify Zoom meetings where your provider workflow allows it. Existing valid links are preserved.

3

Certificates

View provider certificate records, expiry warnings and document status in one place.

Designed to be cautious

TrainerMate is built around “do no harm” behaviour: it verifies before changing, avoids overwriting valid Zoom details, and gives plain-language warnings when manual confirmation is sensible.

Manual confirmation advised

Automated checks can save time, but trainers should still confirm portal details, Zoom links and course joining information before delivery.

Local-first storage

TrainerMate is designed to run on your computer. Provider credentials and Zoom tokens should be stored locally using the operating system credential store where available.
